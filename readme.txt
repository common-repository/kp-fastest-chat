== Description ==

We at <a href="https://www.kreativopro.com/">Kreativo Pro</a> specialize in speeding up WordPress websites. But in 2017 we ran into a new  problem when more and more WordPress users started using Live Chat widgets.

= Why to Use Live Chat? =

According to Neil Patel, Live Chat increases conversion rate by 45%. And we can verify this data ourselves as we use Live Chat on our own WordPress website. Customers just love it!

But Live Chat widgets are known to slow down websites. We speed up thousands of WordPress websites every month and soon we realized that we needed a solution to the Slow Live Chat Widgets.

= Why this plugin? =

Our customers wanted to use Live Chat widgets and still wanted a very fast website. This is when our team got together one Sunday to discuss a possible solution and we found a great idea.

This plugin is a result of that idea. For more than a year, we kept this plugin within our own company and used only for our own customers. But we realized that we could help a lot more WordPress users by just realising it publicly.

And giving back to WordPress community has always been our  objective. So, here we present you our secret idea :)

If you want to use Live Chat and still want a very fast WordPress website, then just use this plugin. It has in-built chat icons and you can use any Direct Live Chat link in it. Almost all good Live Chat widget companies support Direct Chat Link.

= How KP Fastest Cache compares with other chat plugins? =

Chat Widgets Work in Only Two Ways:

* JavaScript Codes (Uses Pings to Communicate from Server to Client)
* Ajax Codes (Uses admin-ajax.php to Communicate from Server to Client)

KP Fastest Chat presents a new way to use Live Chat. It's simple HTML and CSS. No slow JavaScript, Pings and Ajax requests.

The great thing is that our codes can be combined and minified by Popular caching plugins like WP Fastest Cache, Autoptimze, WP Rocket, W3 Total Cache etc.

= What can you do with KP Fastest Cache? =

You can do a lot of things with our plugin. Here are some:

* Use Direct Chat Link and have Live Chat with Your Website Visitors
* Use SMS Link and have visitor mobile phones send you a SMS directly with pre-built message
* Use Call link and have visitors call you direct with one click
* Use Messenger link and have visitors chat with you on their Messenger app
* Use Skype link and have visitors call you or message you on Skype app.
* Use Whatsapp link and have visitors chat with you on Whatsapp with pre-built message on their Whatsapp app.
